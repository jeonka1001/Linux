shell 
- 키보드로 입력받은 명령어를 운영체제(커널) 에 전달하여 그 명령어를 실행하게 하는 프로그램
- 운영체제(커널) 과 사용자 사이의 인터페이스를 감싸고 있기 때문에 '셸(껍데기)' 이라한다.
- CLI(Command-Line-Interface) CUI(Character-based User Interface)

셸 은 명령어를 단어 or 토큰 단위로 쪼개어 해석한다. 

명렁어 예시
date - 시간출력
cal - 달력 출력
uname - 현재 시스템 종류
id - (pc에)로그인한 사용자의 정보 출력
cat 파일명 - 인자로 전달된 파일을 화면에 출력한다.
history (숫자) - 최근 입력한 명령어를 (숫자개) 보여준다
	!(숫자) - 숫자에 해당하는 명령어를 실행한다.
	!! - 직전에 실행한 명령행을 시행한다.

명령어의 구성요소
명령어 + 옵션 + 인자(주로 파일명,경로명)
	^~~ -문자 or --문자열 (일반적으로 -를 제공하며 추가적으로 -- 를 지원하는 명령어도 있다)

/(root) - 최상위 디렉터리
/home - 루트 바로 아래
/home/(계정명폴더) - 실제 홈
. - 해당 경로
.. 해당 경로의 부모 경로

제어키]]
CTRL + C : 인터럽트 발생, 현재 실행중인 프로세스에게 인터럽트 시그널 전송 ( 해당 인터럽트를 받은 프로세스는 종료된다 )
CTRL + D : 현재 실행중인 프로세스에게 EOF를 전송(발생)시킨다.
(bash 에서 입력시 bash가 종료된다 - bash 도 표준 입력으로부터 데이터를 받기 때문에)
CTRL + \ : 현재 "명령어"를 종료(quit) 
CTRL + S : 현재 출력중인 화면을 중지 / CTRL + Q : 현재 중지된 화면을 다시 시작
CTRL + SHIFT + / : 커서 다음 문자를 삭제
CTRL + U : 전체 명령 행 삭제
CTRL + Z : 현재 구동중인 프로그램을 일시 중지(save) - 복원시 fg

단축키]] - 현재 명령 행을 쉽게 편집하기 위해 bash가 제공
CTRL + A(ahead) : 현재 커서 명령행 맨 앞으로 이동
CTRL + E(end) : 현재 커서 명령행 맨 뒤로 이동
CTRL + F(forward) : 현재 커서 다음 한 문자로 이동
ALT + F(forward) : 현재 커거 다음 한 토큰으로 이동
CTRL + B(backward) : 현재 커서를 이전 한 문자로 이동
ALT + B(backward) : 현재 커서 이전 한 토큰으로 이동
CTRL + D : 현재 커서에 위치한 문자를 삭제
ALT + L : 현재 커서로부터 그 토큰의 마지막까지 모두 소문자로 변경
ALT + U : 현재 커서로부터 그 토큰의 마지막까지 모두 대문자로 변경

which 명령어 : 해당 명령어가 있는 위치를 출력한다.
ls 명령어]]
ls -a / --all : 모든 파일 출력 ( 숨김파일까지 )
ls -l : 파일의 자세한 정보를 출력 , 인자를 주면 해당 인자의 자세한 정보 출력
ls -d / --directory : 디렉터리 안의 내용이 아닌 디렉터리 자체 정보 출력 ( 보통 -l 옵션과 같이 사용한다 )
ls -F / --classify : 디렉터리 이름 끝에 '\'를 추가하고 실행 파일이면 * 를 추가하는 등 파일의 종류를 구분하는 문자를 추가하는 옵션
ls -f : 
ls -r / --reverse : 출력을 역순으로 나열
ls -R / --recursive : 현재 디렉터리의 하위 디렉터리 내용을 출력

cat 파일명 : 해당 파일을 실행
cat -n (인자) : 화면에 줄 번호 입력
cat -b (인자) : 비어있지 않은 줄에만 줄 번호 입력
